# 🏍️ Royal Enfield Sales Analytics Dashboard

> An end-to-end Business Intelligence project built using PostgreSQL and Power BI to analyze Royal Enfield sales data through interactive dashboards.

---

## 📌 Project Overview

This project demonstrates how PostgreSQL can be integrated with Power BI to analyze motorcycle sales data and generate business insights. Sales records are stored in PostgreSQL, and Power BI connects directly to the database to visualize updated data after refresh.

---

## 🚀 Features

- 📊 Interactive Power BI Dashboard
- 🗄️ PostgreSQL Database Integration
- ⚡ Data Refresh from PostgreSQL
- 📈 Sales & Revenue Analysis
- 🏍️ Bike Model Performance
- 👥 Customer Analysis
- 🏪 Dealer Performance
- 🌍 State-wise Sales Analysis
- 💳 Payment Mode Analysis

```
     │
     ▼

```

---

## 🔄 Project Workflow

1. Sales data is stored in PostgreSQL.
2. Power BI connects directly to the PostgreSQL database.
3. SQL queries retrieve and transform the required data.
4. Power BI refreshes the dataset.
5. Interactive dashboards provide business insights.
